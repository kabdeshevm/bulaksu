# Bulak Su

Кликабельный статический сайт для проекта практики ТОО «SOFT DRINKS».

## Что внутри

- `index.html` - кликабельный сайт с каталогом, фильтрами, заявкой, калькулятором воды и адаптивной версткой.
- `docs/structure.html` - структура сайта и пользовательский путь.
- `docs/wireframe.html` - скелет главной страницы.
- `docs/moodboard.html` - mood board с палитрой, типографикой и визуальными правилами.
- `screenshots/` - папка для готовых скриншотов этапов.
- `tools/capture-screenshots.ps1` - скрипт для автоматического создания скриншотов.
- `tools/process-product-photos.py` - обработка фотографий 18,9 л и пластиковых бутылок.

## Источники контента

- Старый сайт Bulak Su: https://bulak-su.com/
- Instagram Bulak Su: https://www.instagram.com/bulaksuwater/
- С официального сайта использованы брендовые изображения: логотип, бутылки 0,3 / 0,5 / 0,7 л, водный всплеск и материалы про стеклянную тару.
- По предоставленным изображениям подготовлены товарные фото 18,9 л и пластиковых бутылок.

## Как открыть

Откройте `index.html` в браузере. Сайт не требует сборки и работает как обычная статическая страница.
